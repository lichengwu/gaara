##What's gaara:
1. A `JVM monitoring tool` for HotSpot(later for other JVMs).
2. Provide remote monitoring.
3. i18n supported.
4. Plugin mode, you can choose which parts(jvm,gc,memroy...) to be collected and monitoring.
5. to be continue....
